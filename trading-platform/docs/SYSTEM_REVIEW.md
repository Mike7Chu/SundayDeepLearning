# STOCKLAB 투자 시스템 — 외부 리뷰 요청서

> 이 문서는 개인 투자 플랫폼의 **투자 논리·방법·이론적 근거**를 외부 AI/전문가에게
> 검증받기 위해 작성됐습니다. 시스템이 실제로 구현·가동 중인 규칙을 있는 그대로
> 기술했으니, 논리적 결함·통계적 오류·위험한 가정·개선 우선순위를 지적해 주세요.

---

## 0. 사용자·운영 컨텍스트

- **투자자**: 개인, 실계좌 약 3,400만원. 현재 반도체(SK하이닉스 57% + 삼성전자 39% + 한미반도체 2%)에 96% 이상 집중, 현금 0.2%. 목표는 "코인 투자 손실 복구"(수익률 목표는 사용자가 화면에서 설정).
- **성향(설문 확정)**: 실적+추세 스윙(수주~수개월) · 시스템은 후보 3개+근거 제시, 최종 결정은 본인 · 중립 리스크(종목당 5~10%, 손절 -5~-8%) · 대상은 국내 전체(약 3,600종목)+미국 주요 100종목.
- **운영**: 라즈베리파이4에서 24시간 도커 가동. 시세·잔고는 토스증권 공식 OpenAPI, 국내 재무는 한국투자증권 API(EPS/BPS), 공시·배당·분기실적은 DART, 미국 분기실적은 SEC EDGAR. AI 분석은 Claude(구독 CLI, 주 1회 정기 + 온디맨드).
- **시장 배경(2026.7 실측)**: 코스피 7,000~8,000대, AI·HBM 슈퍼사이클, 삼성전자 31만원대·SK하이닉스 240만원대. 시스템은 "실측 데이터 > AI의 학습 기억" 원칙을 강제함(AI가 옛 가격 기억으로 현재 데이터를 오류 취급하는 사고가 실제로 있었음).

## 1. 데이터 계층과 신뢰 원칙

| 데이터 | 소스 | 주기 | 비고 |
|---|---|---|---|
| 현재가(국내·미국) | 토스 OpenAPI | 15초(관심·보유)/5분(전체) | 가격의 단일 권위 소스 |
| EPS·BPS(국내) | 한국투자증권 | 15초~30분 | PER/PBR/ROE는 토스 가격으로 재계산 |
| 분기 순이익 YoY | DART 정기보고서(국내)/SEC 10-Q XBRL(미국) | 일 1회 | "가장 최신 공식 실적" |
| **잠정실적(공정공시)** | DART 공시 원문 자동 파싱 | 발표 즉시(30초 폴링) | 정기보고서(45일 지연)보다 빠른 매출/영업이익/순이익 YoY |
| 배당 3개년 | DART(액면분할 보정) | 6시간 | 수익률 25%↑ 이상치는 분할 의심으로 숨김 |
| 지수·투자자별 수급 | 토스(코스피/코스닥, 외국인/기관/개인 순매수) | 10분 | |
| ADR | 토스(미국 시세) | 30분 | 괴리율 = ADR$×환율÷(비율×본주₩)−1 |
| 증권사 데일리 리포트 | 사용자가 텔레그램으로 붙여넣기 | 수동 | AI 분석의 최우선 신뢰 입력 |

**신뢰 원칙**: ①AI는 수치를 기억으로 채우지 못하며 미확인="확인 불가", 추론="(추정)" 표기 강제 ②사실과 해석 분리 ③실측 데이터가 AI의 사전지식과 충돌하면 실측이 이긴다.

## 2. 투자 프레임워크 (이론적 기반)

시스템은 4개 학파를 레이어로 결합합니다:

1. **그레이엄 가치**: 그레이엄 넘버 √(22.5×EPS×BPS) 대비 안전마진. 마법공식(그린블라트: 이익수익률+ROE 랭킹)으로 저평가 스크리닝.
2. **멍거 리스크 관리**: "역방향 사고"(이 주식을 지금 사면 망하는 이유만 분석 → 0~30 감점), 능력 범위(데이터 불완전 종목은 거르기), "모르면 사지 않는다"(자산 데이터 없으면 매수 잠금).
3. **추세·모멘텀(스윙)**: MACD(12·26·9) 방향/전환, ADX(14) 추세 강도, SMA20/60 정배열, 이격도 과열 필터. RSI는 강추세에서 랠리 주도주를 '과열' 오판하는 문제로 **의도적으로 제거**함.
4. **실적 모멘텀(PEAD 유사 논리)**: 잠정실적 발표 당일 수치를 점수에 즉시 반영 — 실적 서프라이즈 후 주가가 서서히 반영된다는 가정.

## 3. 의사결정 레이어 (구현된 산식 그대로)

### 3-A. 투자 매력도 점수 (0~100, "살까말까")
- **가치 30**: 그레이엄 넘버 대비 안전마진(+50%에 만점, -50%에 0점 선형).
- **품질 20**: 흑자·ROE≥10%·PBR<3·저부채 체크리스트 비율.
- **성장 15**: 순이익 YoY — 우선순위 잠정실적>분기>연간. -10%=0점, +50%=만점 선형. 데이터 없으면 중립 5점.
- **추세 25**: 정배열·SMA60 상회·3/6개월 모멘텀 체크.
- **타이밍 10**: MACD 상승 힘(5) + 52주 하단 근접(5).
- 판정: 75↑ 적극 검토 / 60↑ 분할매수 / 45↑ 관찰 / 미만 관망.
- 미국 종목은 가치·품질 데이터(EPS/BPS) 부재 → 랭킹에서 제외(왜곡 방지), 개별 조회 시 "성장·추세·타이밍만 반영" 명시.

### 3-B. 2단계 필터 매수리스트 (10분 주기)
1. 정량 필터: ROE>10% AND PBR<1.5 AND PER<15, **데이터 완전 종목만**.
2. AI 역방향 감점: 통과 종목을 Claude가 "망하는 이유만" 분석 → 감점 0~30 (분석 실패 시 보수적으로 30). 유효기간 1주.
3. 최종점수 = 매력도 − 감점, **70점 이상만** 리스트 등재(매수가·손절·목표 동봉).

### 3-C. 오늘의 매매 플랜 (스윙, 10분 주기)
- **1차(차트 불필요, 전 시장)**: 실적 YoY ≥ +10%(잠정 우선) AND 52주 위치 ≥ 50%(추세 프록시) AND 동전주·보유 제외 → s1 = min(YoY,100) + 52주위치×50 + 당일등락 → 상위 40.
- **2차(차트)**: 하드게이트 — 정배열 AND 가격>SMA60 AND **이격도 ≤ SMA20×1.15**(포물선 급등만 추격 금지, 꾸준한 랠리는 통과). 스윙점수 = 실적40 + 추세40(정배열10+ADX≥25면 15+MACD hist>0면 15) + 타이밍20(MACD 골든 직후 20 / SMA20 ±4% 눌림목 14).
- **매수 후보 3**: 근거 + 매수가/손절/목표 + 제안 수량(자산 7.5%와 종목당 한도 중 작은 쪽).
- **매도 점검(보유)**: 심각도 합산 — 손절선 이탈+3, 목표 도달+2, SMA60 이탈+2, MACD 하락+1, 손실 -8% 초과+2, 실적 감소+1. **펀더멘털 상쇄**: 실적 YoY≥+20%면 −2(+"기술 신호와 상충" 명시), 당일 +5% 급등 중이면 −1. 심각도 3 이상만 표시(약한 단일 신호는 소음 처리).

### 3-D. 매매 가격 가이드 (모든 종목·알림 공통)
- 진입: 상승추세면 SMA20 눌림목, 아니면 현재가(추격 매수 방지).
- 손절: 20일 최저가×0.97, 진입 대비 -3%~-15% 클램프.
- 목표: 손익비 1:2 고정(목표 = 진입 + 2×리스크). KRX 호가/센트 반올림.
- 실행: 토스 OCO 조건주문(목표가 익절 + 손절가 손절 서버 감시, 매도 전용)으로 자동화 가능.

### 3-E. AI 아침 점검 (매일 8시, 기관 데일리 노트 형식)
입력: 실계좌 비중·손익, 코스피/코스닥+외국인/기관 순매수, 미국 반도체 6종+빅테크 CAPEX 프록시 5종 간밤 실측, ADR 괴리율, 보유종목 공시·잠정실적, 리스크 실드 상태, 사용자 목표, **사용자 제공 증권사 리포트**(최우선 신뢰).
출력: ①시장·수급(왜+실질영향/노이즈 구분) ②간밤 미국(엇갈림 원인) ③ADR 반영도 ④종목별 판정(계속 보유/일부 매도/위험 신호) ⑤최대 비중 종목 상승/하락 확률(추정, 근거 번호) ⑥성향별 전략(공격/중립/보수) ⑦이 분석이 틀리는 조건 ⑧한 줄 결론.

### 3-F. 리스크 실드 (모든 매수의 상위 게이트)
- MDD 서킷브레이커: 자산 최고점 대비 **-15% → 신규 매수 전면 잠금**.
- 종목당 한도: 자산의 5%(플랜 제안 수량은 7.5% 기준 — 의도적 비대칭).
- 현금 바닥: 현금 비중 25% 미만이면 매수 신호 무시.
- **매도는 항상 허용**(위험 축소 방향은 막지 않음).
- 실주문 4중 게이트: 브로커 키 → 실매매 플래그(기본 꺼짐) → 주문당 금액 한도 → 리스크 실드. 텔레그램 주문은 2분 내 "확인 N" 회신 필수.
- 수급 이벤트: "빛의기둥"(거래대금 20억↑·양봉·고가마감·평소 3배) 전 종목 장중 감지 → 매수/손절/목표 가이드와 함께 알림, 하락추세면 "진입 금지" 경고.

## 4. 알려진 한계 (자체 인지)

1. **백테스트 부재**: 점수 가중치(30/20/15/25/10), 스윙 게이트(이격 15%, ADX 25, YoY +10%), 손익비 1:2는 이론·관례 기반이지 이 시장에서 통계 검증되지 않음. 간단한 SMA/RSI 백테스터는 있으나 플랜 전략 자체의 백테스트는 없음.
2. **확률 제시(⑤)는 AI의 주관적 추정** — 캘리브레이션 검증 없음. "판단 보조용 추정" 명시로만 방어.
3. **펀더멘털 상쇄의 양날**: 실적 좋은 종목의 기술적 붕괴를 "펀더멘털 우위"로 눌러버리면 어닝 피크아웃 하락을 늦게 인정할 위험.
4. **데이터 갭**: 미국 가치·품질(EPS/BPS) 없음, HBM/DDR 현물가 없음(마이크론 주가로 프록시), 종목별 외국인 수급 없음(시장 단위만), ADR 비율은 1:1 가정(사용자 설정으로 보정).
5. **현재 계좌 상태와 규칙의 충돌**: 현금 0.2%라 리스크 실드상 신규 매수가 사실상 불가 — 매수 후보 제시가 실행 불가능한 조언이 되는 구간.
6. **단일 사용자 과최적화**: 규칙들이 반도체 집중 계좌의 최근 사건들(한미반도체 상한가 등)에 반응하며 조정돼 옴.

## 5. 리뷰어에게 — 구체적으로 묻고 싶은 것

1. 매력도 점수 가중치(가치30/품질20/성장15/추세25/타이밍10)가 "실적+추세 스윙" 성향과 정합적인가? 스윙용이라면 가치 비중이 과한가?
2. 스윙 플랜의 하드게이트(정배열+SMA60, 이격≤+15%, YoY≥+10%, ADX≥25 가점) — 임계값이 합리적인가? 놓치는 유형(턴어라운드 초기, 바닥 반전)은 어떻게 보완하나?
3. 손절 산식(20일 저점×0.97, -3~-15% 클램프)과 고정 손익비 1:2 — 변동성(ATR) 기반으로 바꿀 가치가 있나?
4. 매도 점검의 **펀더멘털 상쇄**(-2/-1 심각도 감면)가 어닝 피크아웃 리스크를 과소평가하지 않나? 상쇄에 상한이나 예외(예: 손절선 이탈은 상쇄 불가)를 둬야 하나?
5. AI에게 "상승/하락 확률 %"를 매일 제시시키는 것 — 캘리브레이션 불가능한 숫자가 오히려 과신을 유발하나? 대안 표현이 있나?
6. MDD -15% 서킷브레이커가 **이미 -12% 손실·현금 0.2%인 계좌**에서 실효성이 있나? "물린 풀매수 계좌"에 맞는 리스크 규칙은 무엇인가?
7. 96% 반도체 쏠림 계좌에 대해 시스템이 "분산하라"는 조언과 "반도체 상승 사이클에 올라타라"는 신호를 동시에 내는 모순 — 어떻게 정리해야 하나?
8. 잠정실적 YoY를 발표 당일 성장 축에 즉시 반영(PEAD 유사) — 이미 상한가로 반영된 가격에 점수까지 올리면 이중 반영(추격 유도) 아닌가?
9. AI 역방향 감점(0~30, 실패 시 30)의 보수성 — 분석 실패가 잦으면 매수리스트가 비어버리는 구조인데, 실패 시 대체 규칙이 필요한가?
10. 전체 구조에서 가장 먼저 백테스트/검증해야 할 규칙 하나를 꼽는다면?

> 참고: 모든 출력에는 "투자 판단 보조이며 매매 지시가 아님" 면책이 붙고, 실주문은
> 기본 잠금 상태에서 사용자가 명시적으로 열어야 하며 소액 한도가 강제됩니다.
